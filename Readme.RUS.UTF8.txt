
    Конвертер графических файлов (GRF в BMP).
    -----------------------------------------

       GRF - графический формат файлов Вектор'а-06Ц.
       BMP - графический формат файлов Windows.

    Содержимое.
    -----------

    Каталог 'GIF - Bonus' - Gif-анимация (бонус).
    -------|---
           | Харьков. Разные люди (XX век, 90-е годы).
           | Данная GIF-анимация была сделана из GRF-файлов взятых с Вектор'а-06Ц.
           |---

    Каталог 'SRC C - FreeBSD, Linux' - Исходники на Си.
    Каталог 'SRC C - Windows'        - Исходники на Си.

    Отличие SRC-каталогов - кодировка UTF8 и CP866!

    Сложно ли их скомпилировать исходники? Компиляция в двух действиях!
    Смотрите каталоги 'SRC C - FreeBSD, Linux' и 'SRC C - Windows'
    внутри есть описание.

    Файл 'Graphics Files Vector-06C (GRF).7z'.
    Подборка GRF-файлов (внутри есть и BMP-файлы).
    Название подборки: Харьков. Разные люди (XX век, 90-е годы).

    Примечание.
    Внутри архива ('Graphics Files Vector-06C (GRF).7z') также есть
    программа для просмотра GRF-файлов на Вектор'е-06Ц: программа 'VGRF.COM'
    (для этой программы необходима МикроДОС Т-34).

    Файл 'Info GRF.7z'.
    Как устроен GRF-файл, плюс программа распаковщик (на
    ассемблере, для Вектор'а-06Ц).

    'grf2bmp'     - Исполняемый файл для FreeBSD (GRF-файл в BMP-файл).
    'grf2bmp.exe' - Исполняемый файл для Windows (GRF-файл в BMP-файл).
    'grf2bmp.pl'  - Скрипт на Perl (GRF-файл в BMP-файл).

    Файл 'grf2bmp_Screenshot_1.png'.
    Файл 'grf2bmp_Screenshot_2.png'.
    Скриншоты работы программы.

    Файл 'Readme.RUS.1251.txt' - Это описание в кодировке cp1251 (Windows).
    Файл 'Readme.RUS.UTF8.txt' - Это описание в кодировке UTF-8 (FreeBSD, Linux).
